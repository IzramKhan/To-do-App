# ✅ Python To-Do List App

A simple yet powerful **Python-based To-Do List** that helps you add, delete, check, and save your daily tasks — all directly from the console.  
It’s fast, offline, and written entirely in Python without using any external libraries.

---

## 🚀 Features

- ➕ **Add Tasks** — Quickly add as many tasks as you want.  
- ❌ **Delete Tasks** — Remove tasks that are no longer needed.  
- ✅ **Check Tasks** — Mark tasks as completed.  
- 📋 **View Tasks** — See all, completed, remaining, and deleted tasks.  
- 💾 **Save Tasks** — Save your task list to a `.txt` file for later use.  
- 🧹 **Clear All Tasks** — Instantly remove every task from the system.  
- 🆘 **Help & Description** — Built-in guide and detailed overview of all commands.  

---

## 🧠 How It Works

1. Run the program.  
2. You’ll see a simple menu with options like **Instructions**, **Launch To-Do**, and **Exit**.  
3. Choose **Launch To-Do** to access all task management features.  
4. Type `0` anytime to stop an ongoing action.  

Everything is stored in memory while running, and tasks can be saved to a text file for later reference.

---

## 📂 Files

- `main.py` — The main program file.  
- `tasks.txt` — Automatically created file where all active tasks are saved.
